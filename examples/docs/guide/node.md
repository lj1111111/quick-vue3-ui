## 启动

::: tip 注意
Vite 需要 Node.js 版本 >= 12.0.0。本项目推荐使用版本16.x
:::

### 安装使用

- 克隆项目

```bash
git clone https://gitee.com/zhanglp520/quick-node-express.git
```

- 安装 pnpm

```bash
npm install -g pnpm
```

- 安装项目依赖包

```bash
pnpm install
```

- 开发环境下运行命令

```bash
pnpm run dev
```

- 生产环境下运行命令

```bash
pnpm run prod
```

运行命令执行服务启动后，浏览器输入：localhost:3102
