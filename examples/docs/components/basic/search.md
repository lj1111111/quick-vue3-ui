## 属性

| 属性名            | 描述                 | 类型            | 默认值 | 可选值 |
| :---------------- | :------------------- | :-------------- | :----- | :----- |
| model             | 搜索表单 model       | Object          | -      | -      |
| items             | 搜索表单项集合       | Array<FormItem> | -      | -      |
| searchButtonName  | 自定义搜索按钮文本值 | String          | 查询   | -      |
| resetButtonName   | 自定义重置按钮文本值 | String          | 清空   | -      |
| hiddenResetButton | 隐藏重置按钮         | Boolean         | false  | -      |
